# 🎯 Number Guessing Game

A simple Python command-line game where the player tries to guess a secret number. The game provides hints after each attempt and tracks the total number of guesses required to win.

## ✨  Features

- Interactive user input
- High/Low guessing hints
- Attempt counter
- Beginner-friendly Python implementation

## 🛠️ Technologies Used

- Python 3

## ▶️ How to Run

1. Clone the repository:

```bash
git clone https://github.com/Jiya-Choudhary/Number-Guessing-Game.git
```

2. Navigate to the project folder:

```bash
cd Number-Guessing-Game
```

3. Run the program:

```bash
python guessing_game.py
```

## 📸 Sample Output

```text
Enter a number: 150
Number is Low

Enter a number: 250
Number is High

Enter a number: 200
Number of Attempts: 3
You Win!
```
